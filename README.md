# Forecast Eval TeachIn

## Setup Instructions

### 1. Create a Conda Environment

```bash
conda create -n forecast-eval-teachin
conda activate forecast-eval-teachin
```

### 2. Install pip (if not already installed)

```bash
conda install pip
```

### 3. Install Requirements

```bash
pip install -r requirements.txt
```

### 4. Run the Demo

```bash
python demo.py
```

## Project Structure

- `demo.py` - Main demo script
- `colours.R` - BoE colours
- `slides.md` - Presentation slides
- `requirements.txt` - Python dependencies
- `images/` - Image assets
